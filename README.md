# Lista.ai

O **Lista.ai** é um projeto em construção voltado para o gerenciamento inteligente de listas de compras.

A proposta do sistema é permitir a criação e organização de listas de forma manual, além de incorporar sugestões geradas por IA a partir de receitas e suportar fluxos de compartilhamento entre usuários.

A arquitetura do projeto está sendo construída com foco em aprendizado e aplicação prática de conceitos como:

- microsserviços
- arquitetura hexagonal
- DDD tático leve
- comunicação assíncrona com Kafka
- integração com IA

## Estrutura planejada

- `list-service` → serviço principal responsável pelas listas e itens
- `notification-service` → responsável pelo fluxo de compartilhamento e notificação
- `recipe-service` → responsável pela geração de ingredientes com apoio de IA
